2017-05-30 GIT

今天终于把github简单设置成了。大概就是以下几步：

  • 建一个本地目录，进入，然后初始化。(user.name,user.email,init)
  • 进入gitbash，生成ssh key. 然后在github上加入。
  • 添加远端库。(git remote add )
  • push 或 clone(push 之前要add + commit)

一切才开始。
